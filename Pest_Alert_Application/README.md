# Pest_Alert_Application
STM32-Pest_Alert 시스템을 통해 블루투스 모듈이 보낸 신호를 
안드로이드 스마트폰에서  확인할 수 있는 애플리케이션이다.

## Tested Devices
 - Galaxy S20+
 - Galaxy  Note8
